# Setting Up Deep Links

Check the repo shape first.

- If the app uses Expo config or Expo dependencies, read `references/expo.md`; otherwise read `references/native.md`.
- Then inspect the navigation style.
- If the app uses static config, read `references/static.md`.
- If the app uses JSX navigation, read `references/dynamic.md`.
- Load exactly one environment reference and one linking reference.
- Do not load both refs from either pair unless explicitly comparing approaches.
