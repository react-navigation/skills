# Setting Up TypeScript

Check `@react-navigation/native` in `package.json` and inspect the navigator style first.

- If the navigator tree is static, read `references/static-react-navigation-7.md` for `7.x` or `references/static-react-navigation-8.md` for `8.x`
- If the navigator tree is dynamic, read `references/dynamic.md`
- Load exactly one reference file
- Do not load both static references unless you are explicitly comparing versions
