# React Navigation Skills

Official AI agent skills for React Navigation.

This repository contains reusable skills for agent clients that support the Agent Skills format, including Claude Code, Codex, and similar tools.

## Skills

- `migrate-to-static-config`
  Migrate React Navigation navigators from dynamic JSX config to static config, with version-specific guidance for React Navigation 7.x and 8.x.
- `upgrade-react-navigation`
  Upgrade React Navigation from 6.x to 7.x or from 7.x to 8.x, with version-specific migration guides.

## Compatibility

These skills are intended for clients that understand the Agent Skills format. Each skill lives in its own directory under `skills/`.
